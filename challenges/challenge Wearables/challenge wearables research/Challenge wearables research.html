Shy Mask That Shuts Up When It Sees People

In dit project heeft iemand een gerobotiseerd mondmasker gemaakt waarbij het klepje over de mond dichtgaat als een van de sensoren een beweging waarneemt. De maker vond de mondmaskers die nu gedragen moeten worden tijdens de Covid-19 periode niet lekker zitten. Ook wilde de maker graag wat kunnen drinken door een rietje terwijl deze een mondmasker op heeft. Daarom begon hij dit project.

Ik vind dit project inspirerend, omdat het laat best uitgebreid is. De maker heeft onderdelen gemaakt met een 3D-printer, gesoldeerd en een soort open- en sluitmechanisme gemaakt. Tevens heeft hij sommige onderdelen beschilderd. Door dit gave project heb ik inspiratie op kunnen doen wat mogelijk is met een Arduino.

Maker: Chen the Design Maker.

Mind Control Drone

In dit project is een ‘Mind Control Drone’ gemaakt, oftewel een drone die met je gedachten te besturen is. Bij dit project is onder andere een Arduino, kleine drone en een ‘MineWave sensor’ gebruikt waarmee de drone te besturen is. De maker geeft aan dat deze drones fascinerend vind. Toen hij de kans kreeg om een project te doen met ‘coole technologie,’ besloot deze een met gedachten bestuurbare drone te maken.

Ik vind dit project erg vet, omdat ik niet wist dat je met simpele spullen een drone kan maken die met je gedachte te besturen is. Het klinkt namelijk best indrukwekkend. Volgens de maker komt de drone omhoog als je ‘concentratieniveau’ boven de 40 is. Ik weet niet precies hoe dit in zijn werking gaat, maar vind het wel gaaf. Door dit project ben ik mij bewust geworden dat je geen dure spullen nodig hebt om iets gaafs te creëren.

Maker: Wesley Hartogs.

Musical Shoes

In dit project heeft iemand muzikale schoenen gemaakt. De maker kwam erachter hoe deze geluiden kan laten ontstaan als een bepaalde beweging wordt uitgevoerd. Omdat de maker ook van muziek houdt, besloot deze dit project te beginnen.

Ik vind dit project inspirerend, omdat het er niet heel moeilijk uitziet. Een compact doosjes buiten de schoen is gecombineerd met sensoren in de schoen. Samen zorgen deze ervoor dat er een geluid ontstaat als een bepaalde beweging met de voet is gemaakt. Ik vind dit een grappig project dat een leuke toepassing is voor dansers, zo kunnen deze dansend een liedje maken. Erg inspirerend dus.

Maker: Lara Grant
